# python-piscine

A collection of solutions for the Python Piscine curriculum, covering core Python concepts through progressively more advanced programming exercises. The repository emphasizes clean code, object-oriented design, data manipulation, decorators, closures, dataclasses, and statistical computations while relying solely on Python's standard library where required by the subject.

## Module 0 – Introduction

Fundamentals of Python programming, including syntax, functions, data types, control flow, string manipulation, error handling, and basic scripting practices.

## Module 1 – Arrays

Working with Python collections, list operations, iterators, statistical calculations, higher-order functions, closures, decorators, and custom algorithms without external libraries.

## Module 2 – Data Tables and Graphs

Data processing using tabular datasets, data analysis techniques, aggregation, filtering, visualization, and plotting to extract insights from structured data.

## Module 3 – Object-Oriented Programming

Core OOP principles in Python, including classes, inheritance, abstract base classes, multiple and diamond inheritance, properties, operator overloading, decorators, dataclasses, and encapsulation.

## Module 4 – Data-Oriented Design

Advanced Python features with an emphasis on decorators, closures, type hints, dataclasses, object lifecycle, serialization concepts, and writing maintainable, reusable components.
